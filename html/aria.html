ARIA

Issue

Sometimes, semantic HTML is not descriptive enough. For instance, there is not a native HTML element to describe the icon in the top left of the screen.

Solution

Add ARIA-labels or roles to provide further context.

Exercise

Write an aria-label describing the functionality of the navigation menu icon in the top left corner.

Your answer:

Sample answer: aria-label="Navigation menu"

Press "n" to go back or "m" to go continue.
