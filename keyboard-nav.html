Keyboard Navigation

Issue

For various reasons, some users cannot use a mouse.

Solution

Ensure all features on the page can be accessed via keyboard commands.

Exercise

Use the “Tab” key to highlight the Light Mode toggle, then "Spacebar" to toggle between light modes.

Press "n" to go back or "m" to continue.
